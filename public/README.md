# Images Directory

Place your images here to display them on your site:

- `profile.jpg` - Your profile photo (will be displayed as a circle)
- `project1.jpg` - First project screenshot (Guided by Value)
- `project2.jpg` - Second project screenshot (Trick or Treat)
- `project3.jpg` - Third project screenshot (NFT Project)

The images will automatically display on the homepage. If images are not found, placeholder colored boxes will be shown instead.
